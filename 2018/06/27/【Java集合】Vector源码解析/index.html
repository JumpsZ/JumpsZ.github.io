Vector

数据结构

  • Vector 是矢量队列，它是JDK1.0版本添加的类。继承于AbstractList，实现了List, RandomAccess, Cloneable这些接口。
  • 继承于AbstractList，是一个队列，支持相关的添加、删除、修改、遍历等功能
  • 实现了RandmoAccess接口，即提供了随机访问功能。
  • 线程安全，不需要线程安全，则使用ArrayList

总结
